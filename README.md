# Minecraft Art
A program which turns pictures into Minecraft pixelart.

## Flags
| Flag | Description |
|-|-|
| *-s* | Scale, i.e., how shortened output image is, compared to the input image. The scale is equal to 1 by default. If the scale is equal to `n` each `n`Ã—`n` fragment of the input image will be attributed to one block. |
| *-o* | Name for the output image. |
| *-v --version* | The program version. |
| *-h --help* | The information you're reading right now. |
